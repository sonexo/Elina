# Elina
Bot Discord for Development

### How to Use
- add your `bot token`, `mongodb`, `top.gg` in `.env` 
- run `npm i elina`
- run node `index.js`
### How to Bot Online 24/7
- You can use uptime robot
- or you can also use the google app script to monitor the bot project.
